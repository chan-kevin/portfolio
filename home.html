The Adventures of Gary the Snail

A 2D interactive puzzle game using only javascript

The Adventures of Gary the Snail is a 2D interactive puzzle game where Gary the snail is lost underwater. The main objective of the game is to help Gary get to home to Spongebob in Bikini Bottom. There are 6 frames in a 2x3 formation in every scene which can be moved around to help Gary get from the starting frame to the frame where the bubble is at.

Each bubble leads to the next scene/level. Gary moves in one fixed direction until he hits an obstacle, where he will either lose a life (jellyfish) or start moving in the opposite direction (rock). The game intends to introduce functionalities slowly to allow users to get the hang of it. There will be approximately 3-5 levels in this game.

Skills: Canvas API · JavaScript · Cascading Style Sheets (CSS)
