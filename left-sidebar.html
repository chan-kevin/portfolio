The Adventures of Gary the Snail

A 2D interactive puzzle game using only javascript

Background

The Adventures of Gary the Snail is a 2D interactive puzzle game where Gary the snail is lost underwater. The main objective of the game is to help Gary get to home to Spongebob in Bikini Bottom.

  • Play Now

Gameplay

Users will be able to click on two tiles and swap the objects inside. The objective is to guide Gary to SpongeBob's home/bubble. However, if a jellyfish is hit, it will result in a game over.

Introduction

There are 6 frames in a 2x3 formation in every scene which can be moved around to help Gary get from the starting frame to the frame where the bubble is at. Each bubble leads to the next scene/level. Gary moves in one fixed direction until he hits an obstacle, where he will either lose a life (jellyfish) or start moving in the opposite direction (rock). The game intends to introduce functionalities slowly to allow users to get the hang of it. There will be 4 levels in this game.

Functionalities

In The Adventures of Gary the Snail, user will be able to:

  • Swap the position of two frames by clicking consecutively on both frames.
  • Obstacles that cause the character to lose their lives will result in the game restarting.
  • Upon touching bubble, Gary floats away in it and next level starts
  • Upon reaching Spongebob's house (pineapple) in final level, Spongebob appears
  • A rule page will also be provided to the users
